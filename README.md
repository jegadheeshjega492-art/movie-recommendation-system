# Movie Recommendation System

A Machine Learning-based Movie Recommendation System developed using Python, Scikit-learn, Pandas, and NLP techniques. The system provides personalized movie recommendations using content-based and collaborative filtering methods.

---

## Features

- Content-based movie recommendation
- Collaborative filtering recommendation
- TF-IDF vectorization
- Cosine similarity matching
- Personalized top-10 recommendations
- Fast and efficient recommendation engine

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLP
- TF-IDF
- Cosine Similarity
- Jupyter Notebook

---

## Dataset

- 5,000+ movie dataset
- Includes movie titles, genres, keywords, cast, and overview

---

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Extraction
4. TF-IDF Vectorization
5. Cosine Similarity Calculation
6. Recommendation Generation
7. Evaluation using Precision@10 and Recall@10

---

## Output

- Achieved 85% recommendation relevance
- Generates top-10 personalized movie recommendations
- Tested across 200 user profiles
